🚀 Getting Started

1 Add Logo

Place your logo_chitose.png in the assets/ folder

Recommended: 256x256px PNG with transparent background

2 Open Pages

Navigate through the pages using the dashboard or use direct links:

  • • pages/dashboard.html
  • • pages/input.html
  • • pages/master-rekap.html

3 Customize

Edit HTML files to customize content, colors, and data as needed

All pages use Tailwind CSS for easy styling

4 Deploy

Upload all files to your web server or hosting platform

Fully static - no backend required!
